# MYHEALTH

## BACKEND

How to run?

- Config your database in `src/configs/db.json` and `.env`
- Run this commandline at the root of project:

```bash
    # install all packages
    npm install

    # Migrate database
    npm run migrate

    # Seed
    npm run seed

    #Run
    npm run dev

    ---------------
    # More:
    # Undo migrate
    npm run undo-migrate

    # Undo seed
    npm run undo-seed
    
```
